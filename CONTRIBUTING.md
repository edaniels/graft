# Contributing to Graft

Thank you for your interest in contributing to Graft!

## AI Usage Policy

This project uses AI tools responsibly as part of its development workflow. **All code is human-reviewed prior to being merged**, regardless of whether it was written by a human, assisted by AI, or generated by AI.

### Disclosure requirement

All contributors must disclose AI tool usage in **every pull request and issue**. This applies to all contributions, no matter the author. Specifically:

1. **Pull requests**: Use the AI disclosure section in the PR template to indicate whether AI tools were used, which tools, and how they assisted.
2. **Issues**: Use the AI disclosure section in the issue template to indicate whether AI tools assisted in preparing the issue (e.g., debugging analysis, research, drafting).

### What counts as AI usage

**Disclosure required:**
- Code generation, refactoring, or debugging assistance
- Documentation or prose drafting
- Test generation
- Architectural or design suggestions that were adopted
- Analysis used to prepare an issue or review

**No disclosure needed:**
- IDE autocomplete (single-line completions)
- Spell check or grammar correction
- Search engine queries

### Accountability

The submitting contributor is fully responsible for every contribution, whether a human wrote it or AI assisted. "The AI generated it" is not an acceptable explanation for bugs, security issues, or licensing problems. Contributors must:

- **Understand** all code they submit
- **Test** AI-assisted contributions with the same rigor as hand-written code
- **Verify** that AI-generated code does not introduce licensing conflicts

### Review standards

All contributions receive the same quality review regardless of origin. Maintainers may apply additional scrutiny to disclosed AI-assisted contributions where appropriate.

## Development

See the [README](README.md) for build, test, and lint commands.

### Testing

We practice test-driven development. All changes should try to include tests.

### Code style

Run `just lint` before submitting.
